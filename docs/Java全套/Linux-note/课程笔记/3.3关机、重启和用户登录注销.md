# 关机、重启和用户登录注销

## 1. 关机&重启命令

### 1.1 shutdown

* shutdown -h now：表示立即关机
* shutdown -h 1：表示1分钟后关机
* shutdown -r now：立即重启

### 1.2 halt

* halt：直接使用，关机

### 1.3 reboot

* reboot 重启系统

### 1.4 sync

* sync：把内存的数据同步到磁盘上

> 当我们关机或者重启时，都应该先执行一下sync，防止数据丢失。

## 2. 用户登录注销

* 登录时少用root账号登录，以避免操作失误的损失
* 平时可登录普通账户，再用“su - 用户名”切换成管理员登录

### 2.1 logout

* logout：注销用户

> 在图形运行级别无效，在运行级别3有效。
